Question

I am using Git on Windows 7 and access my repositories through Git Bash. How can I change the default location that Git Bash opens when I start it? It's somewhat time consuming to navigate to htdocs, and then a specific folder. Is there a way to change the configuration file to have it open elsewhere? Or would it be possible to write a .sh file to do this? Unfortunately Git Bash won't open my htdocs folder shortcut on my desktop, and it takes 5 cds to get to my desired directory.

  • ACCEPTED

    After installing msysgit I have the Git Bash here option in the context menu in Windows Explorer. So I just simply navigate to the directory and then open Bash right there I also copied the default Git Bash shortcut to the desktop and edited its Start in property to point to my project directory. It works flawlessly. Unfortunately Git Bash won't open my htdocs folder shortcut on my desktop, and it takes 5 cds to get to my desired directory.
